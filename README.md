## Deploy to Railway
1. Fork this repo to your GitHub
2. Railway.app → New Project → Deploy from GitHub
3. Add Variables:
   - MONGODB_URI = (MongoDB Atlas free link)
   - OWNER_NUMBER = 9477xxxxxxx
4. Deploy!

Website auto runs at your Railway URL
